# QA Checklist (Manual, 15 tests)

Дата: `____-__-__`  
Тестер: `__________`  
Версия бота: `python -m bot.main` (локальная)

## Правила отметки
- `PASS` - поведение совпало с ожидаемым.
- `FAIL` - не совпало (добавь короткий комментарий).

## Тесты
1. `/start` у нового пользователя  
   Ожидаемо: приветствие, меню, список игр, стартовый бонус `$0.10` начислен один раз.  
   Результат: `PASS/FAIL` Комментарий: `_____`

2. Повторный `/start` тем же пользователем  
   Ожидаемо: бонус `$0.10` повторно не начисляется.  
   Результат: `PASS/FAIL` Комментарий: `_____`

3. Нижнее меню (визуал)  
   Ожидаемо: сверху `Играть` + `Пополнить` (зелёные), остальные кнопки без заливки.  
   Результат: `PASS/FAIL` Комментарий: `_____`

4. Экран выбора игр  
   Ожидаемо: есть `Кости`, `Футбол`, `Баскетбол`, `Слоты`, `Мины`, `Краш`, `Рулетка`; нет `Дартс/Боулинг`.  
   Результат: `PASS/FAIL` Комментарий: `_____`

5. Кости: выигрышная ставка  
   Шаг: поставить на исход, который выпал.  
   Ожидаемо: выплата больше ставки, множитель в кнопке соответствует фактической выплате.  
   Результат: `PASS/FAIL` Комментарий: `_____`

6. Кости: проигрышная ставка  
   Ожидаемо: списывается 100% ставки, финальный текст корректный.  
   Результат: `PASS/FAIL` Комментарий: `_____`

7. Слоты: не-777  
   Ожидаемо: проигрыш, текст "выигрывает только 777 (x10)".  
   Результат: `PASS/FAIL` Комментарий: `_____`

8. Слоты: 777  
   Ожидаемо: только при 777 выплата `x10` от ставки.  
   Результат: `PASS/FAIL` Комментарий: `_____`

9. Краш: старт и обратный отсчёт  
   Ожидаемо: предупреждение + 5 секунд до старта, затем активная кнопка `Забрать`.  
   Результат: `PASS/FAIL` Комментарий: `_____`

10. Краш: ручной кэшаут  
    Ожидаемо: при нажатии `Забрать` до взрыва выплата зачисляется, итоговый экран корректный.  
    Результат: `PASS/FAIL` Комментарий: `_____`

11. Краш: поздний кэшаут / взрыв  
    Ожидаемо: при взрыве ставка теряется, отображается точка взрыва.  
    Результат: `PASS/FAIL` Комментарий: `_____`

12. Мины: открытие клеток и кэшаут  
    Ожидаемо: множитель растёт, кэшаут работает, при мине полный проигрыш ставки.  
    Результат: `PASS/FAIL` Комментарий: `_____`

13. Пополнение: экран методов  
    Ожидаемо: есть CryptoBot, есть Stars, указан курс `1⭐ = $0.017` и `100⭐ = $1.70`.  
    Результат: `PASS/FAIL` Комментарий: `_____`

14. Stars инвойс и зачисление  
    Шаг: оплатить Stars.  
    Ожидаемо: в тексте видно сколько `$` будет зачислено, после оплаты баланс увеличен в USD по курсу.  
    Результат: `PASS/FAIL` Комментарий: `_____`

15. Рефералка + повтор ставки + поддержка  
    Ожидаемо:  
    - `/ref`: указано `10%` от проигрышей приглашённых;  
    - после игры есть `Повторить ставку`;  
    - `/support` и кнопка в меню открывают контакт техподдержки.  
    Результат: `PASS/FAIL` Комментарий: `_____`
