5) CRASH
5.1 Типы исходов
- Ручной кэшаут до взрыва - динамический x
- Взрыв до кэшаута - 0x

5.2 Формула выплаты
- Эффективный множитель: effective_x = 1 + (raw_x - 1) * (1 - edge)
- При текущем edge 0.22: effective_x = 1 + (raw_x - 1) * 0.78

5.3 Примеры
- raw 1.10 -> 1.08x
- raw 1.50 -> 1.39x
- raw 2.00 -> 1.78x

5.4 Распределение точки взрыва (raw множитель)
- 18%: 1.01-1.05
- 27%: 1.06-1.20
- 30%: 1.21-1.80
- 16%: 1.81-3.00
- 7%: 3.01-4.80
- 2%: 4.81-8.00

7) МИНЫ

7.2 Исходы

7.3 Формула множителя
- fair = C(25, n_safe) / C(25 - bombs, n_safe)
- adjusted = max(1.00, fair * (1 - total_edge))
- total_edge = house_edge_mines + extra_edge
- extra_edge:
  - bombs <= 3  -> +0.17
  - bombs <= 5  -> +0.10
  - bombs <= 7  -> +0.05
  - bombs >= 8  -> +0.00
